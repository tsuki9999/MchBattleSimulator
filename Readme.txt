MchBattleSimulator は、マイクリプトヒーローズ My Crypto Heroes( https://www.mycryptoheroes.net/ja )の戦闘シミュレータです。

実行ファイルに同梱されたheroes.txtからヒーローの編成を読み込んで、指定した回数のバトルを繰り返し、勝ち数と負け数を表示します。
また、最初の10戦の戦闘ログをbattle_log_(ファイル番号).txtに書き込みます。

シミュレートするチームの編成は、以下の編成用Webページで行ってください。
https://tsuki9999.github.io/MchBattleSimulator/heroes.html

このページでチームを編成すると、ページ下部の欄に数字が表示されるので、それをすべてコピーしてheroes.txtに貼り付けてください。


もしくは、Googleスプレッドシートで編成することもできます。
https://docs.google.com/spreadsheets/d/1ISTTGVUYv-ZZH2bnV84pdSNAq61ZH436KLaBeUh4lb4/edit?usp=sharing

使い方の詳しい説明は https://tsuki9999.github.io/MchBattleSimulator/ にあります。

動作確認は Windows 10 64bit OS で行いました。

不具合などのご連絡は、twitter ID @luteola9999　もしくは　luteola9999@gmail.com までよろしくお願いします。
